Dark Mode System
Semua warna menggunakan CSS variables yang otomatis beradaptasi. Tidak ada hardcode hex di komponen.
Strategy
CSS variables otomatis via prefers-color-scheme. Semua komponen inherit — tidak ada overriding manual per komponen.
Surface hierarchy (dark)
Level 0: near-black background
Level 1: card surface (lighter)
Level 2: input / secondary fill
Level 3: hover state
Never hardcode
Jangan gunakan #333, #fff, atau rgba(0,0,0,...) langsung. Selalu gunakan var(--color-*)
Gradient adjustment
Gradient opacity dikurangi di dark mode. Gunakan prefers-color-scheme: dark media query jika perlu tuning.
Color stop usage (light vs dark)
Accessibility Guidelines
WCAG 2.1 AA minimum. Target AA+ untuk teks kritis (jurnal, insight). Emosi adalah kebutuhan — akses adalah hak.
